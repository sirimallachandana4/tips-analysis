# Restaurant Tips Analysis

Exploratory data analysis of restaurant tipping patterns using Python, Pandas, and Matplotlib.

## What I explored
- Which day of the week gets the highest average tips
- Whether total bill amount predicts tip size (correlation analysis)
- How tipping differs between lunch and dinner service

## Key findings
- Sunday has the highest average tip ($3.26)
- Strong correlation (0.68) between bill size and tip amount
- Dinner tips are consistently higher than lunch

## Tools used
- Python 3.10
- Pandas — data loading, groupby, filtering
- Matplotlib — bar charts, scatter plots, histograms
- Jupyter Lab — interactive notebook environment

## How to run
1. Clone this repo
2. Install dependencies: `pip install pandas matplotlib jupyter`
3. Open `tips_analysis.ipynb` in Jupyter Lab
